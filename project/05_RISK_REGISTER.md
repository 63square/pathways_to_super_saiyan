# 05 — Risk Register (v1.1, Post-SAB Review)

## Methodology

Each risk is scored on:
- **Probability** (1–5): 1 = <5%, 2 = 5–20%, 3 = 20–50%, 4 = 50–80%, 5 = >80%
- **Severity** (1–5): 1 = nuisance, 2 = minor, 3 = phase delay, 4 = phase failure, 5 = program-ending or life-threatening
- **Risk Score** = P × S

Risks scoring ≥12 require active mitigation plan with named owner. Risks ≥16 require Board-level monitoring with quarterly Board review.

## 5.5 Quantitative VaR (NEW per SAB)

Monte Carlo over the risk register with correlation matrix (10,000 trials, 5-year horizon):

| Metric | Value |
|---|---|
| **Expected loss** (mean) | $1.2B |
| **VaR(95%)** | $3.8B |
| **VaR(99%)** | $6.2B |
| **Expected Shortfall beyond 99%** (CVaR) | $8.5B |

**Principal personal balance-sheet stress test:** at $6B cumulative drawdown, the Principal's other enterprises may be impaired (depending on capital structure). The Phase 1+ funding plan must include explicit balance-sheet headroom analysis.

## 5.6 Reputational Contagion Map (NEW per SAB)

The program's reputational risk extends beyond the program itself. Required: enumerate Principal's other operating companies, public-facing brands, board seats, political relationships. For each, map the contagion path from a program scandal.

- **Information firewall** between program entities and other Principal ventures (no shared counsel, auditors, banks, PR firms)
- **Cross-entity directorship minimization**
- **Pre-agreed disclosure protocol** to each other venture's board on defined triggers (Grade 3+ AE, regulatory action, press inquiry above threshold)
- **Joint reputation insurance / crisis-comms retainer** sized to *aggregate* exposure

Contagion map is privileged document held by GC + CSecO; updated quarterly.

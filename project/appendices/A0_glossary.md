# Appendix A0 — Glossary

## Program Terms

| Term | Definition |
|---|---|
| **Principal** | The subject of the program; also the funder in the n=1 self-experimentation model |
| **PMO** | Project Management Office; central program coordination |
| **PROMETHEUS** | Program codename; never used in external materials |
| **Holdco** | Cayman parent entity owning IP and spinout equity |
| **Próspera entity** | Honduras-domiciled operating subsidiary |
| **Phase** | Major program stage (0 through 4) with distinct objectives and budget |
| **Gate** | Binding go/no-go decision point with documented criteria |
| **Tranche** | Capital release sub-unit within a phase |
| **KPI** | Key Performance Indicator; pre-registered measurable outcome |
| **SAB** | Scientific Advisory Board; 9 external members |
| **Ethics Council** | Internal IRB-equivalent within Próspera jurisdiction |
| **Spinout** | A company formed to commercialize a specific program-generated technology |

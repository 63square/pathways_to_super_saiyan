## 4.8 Sensitivity Analysis

Top three variables (Monte Carlo sensitivity):

1. **Phase 0 success probability** (estimated 60–75%). If Phase 0 fails outright, program halts at $62M sunk. Spinout proceeds in pure-failure: $50–200M. **Phase 0 cost is fully recovered even on outright failure.**
2. **Próspera + UAE + El Salvador joint failure** (probability all three lost: ~3%). Recovery via Cayman + research vessel logistics ~$80M relocation + 12–18 month delay.
3. **Talent inflation** (15–25% annually). Could push Phase 1+ personnel by +30% over forecast.

## 4.10 Tax Optimization (NEW per SAB)

For US-person Principal (assumed):

1. **CFC + GILTI exposure** on Cayman Holdco operating income — annual tax, not deferred. Mitigation: structure operating subsidiary in jurisdiction with high-tax exception (Cayman with effective tax via Próspera/UAE entities) **or** restructure to US C-corp for §1202 QSBS stacking.
2. **§1202 QSBS** on US-domiciled spinout C-corps: $10M federal-tax-free gain per shareholder per spinout. Across 12 spinouts that's potentially $120M+ tax savings. **Action:** structure US-domiciled spinouts as QSBS-eligible C-corps where feasible.
3. **PFIC** on Cayman-domiciled spinouts: avoid by structuring spinouts as Delaware C-corps with Cayman holding only IP licensing royalty stream.
4. **Expatriation** option (§877A exit tax) before major appreciation — tax counsel to advise pre-T1.
5. **Charitable LLC / DAF** structures for portion of exit proceeds — defers/eliminates tax on charitable portion while supporting program adjacent research.

**Estimated lifetime tax efficiency value: $300M–$1B+ depending on exit outcomes.**
